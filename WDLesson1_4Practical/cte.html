Computer Programming and Web Design

Computer Programming and Web Design (CPWD) is a Career and Technical Education (CTE) program offered at Bayside High School in Queens, New York. The program provides students with the skills and knowledge needed to pursue careers in computer programming, web design, and related fields. Students in the CPWD program learn a variety of programming languages, including HTML, CSS, JavaScript, Python, and Java. They also learn about web design principles, user experience (UX) design, and project management. The program emphasizes hands-on learning and real-world applications, with students working on projects that simulate real-world scenarios. Graduates of the CPWD program are well-prepared to enter the workforce or pursue further education in computer science or related fields.
